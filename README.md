## Quick start <br/>
**1. clone this repository**<br/> 
```console 
$ git clone https://github.com/JCharls1/TicTacToe-MiniMax.git
```
**2. Open git bash terminal**<br/> 
**3. Run the code with your language of choice**<br/> 
c++
```console 
$ chmod +x run.sh
$ ./run.sh c++
```
java
```console 
$ chmod +x run.sh
$ ./run.sh java
```
python
```console 
$ chmod +x run.sh
$ ./run.sh python
```
